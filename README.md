# ChessBot ♟️

ChessBot is a computer vision–powered chess analyzer that detects board positions from images or video streams and suggests the best move.  
Built with **Python, Java, OpenCV, and Selenium**, this project combines image processing and automation techniques to analyze and interact with chess games.

---

## 🚀 Features
- Detects chess pieces on a real board using **OpenCV**.
- Converts detected positions into FEN notation.
- Suggests best moves using chess engines (e.g., Stockfish).
- Automates interaction with online chess platforms using **Selenium**.
- Supports image preprocessing (thresholding, edge detection) for improved accuracy.

---

## 🛠️ Tech Stack
- **Languages**: Python, Java
- **Libraries & Tools**: OpenCV, Selenium, NumPy
- **Other**: JUnit (testing), Git, VS Code
